# Настройка базы данных Supabase

Так как вы создали абсолютно новую базу данных, она пустая. Чтобы мессенджер заработал, нужно создать таблицы, настроить правила доступа (RLS) и хранилища (Buckets).

Я написал полный SQL-скрипт, который сделает всё это за одну секунду.

## Пошаговая инструкция:

1. Откройте панель управления вашей базой данных Supabase: [https://supabase.com/dashboard/project/](https://supabase.com/dashboard/project/)
2. Перейдите в раздел **SQL Editor** (иконка консоли `>_` в левом меню).
3. Нажмите кнопку **New query**.
4. Откройте файл `SUPABASE_SETUP.sql` в этом проекте и **скопируйте ВЕСЬ его код**.
5. Вставьте скопированный код в SQL Editor в Supabase.
6. Нажмите кнопку **Run** (или `Cmd+Enter` / `Ctrl+Enter`).
7. Дождитесь зеленой надписи "Success".

**Всё готово!** 
База данных, триггеры для правильных никнеймов (теперь не будет "user"), хранилища для файлов, голосовых и видео-кружочков настроены и готовы к работе.
